INSTALLATION DES TEMPLATES :

- Dans chaque dossier vous verrez le fichier .zip ou .rar portant le nom du template
- Certains dossiers comprennent une documentation qui peuvent vous être utile
- Depuis Wordpress => Theme => Transverser thème. Déposer le fichier .zip ou .rar
- Activer le thème puis dans personnaliser, vous pouvez selectionner le modèle.
