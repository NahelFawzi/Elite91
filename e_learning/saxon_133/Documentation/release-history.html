Additional info

Theme folder: /wp-content/themes/saxon/

Plugins folder: /wp-content/plugins/

*.* - mean ALL files with ANY types inside specified directory

*.php - mean ALL files with .php type inside specified directory

The best way to update plugins manually is replace all plugin folder files with files from new plugin version archive by FTP. Some plugins (like Visual Composer) may require deactivation and activation again for complete update.
